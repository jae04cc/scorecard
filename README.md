# 🎮 Game Night Scorecard

A simple, responsive web app for tracking scores across multiple games.
Built for real-world game nights and designed to run in Docker
(with Unraid compatibility in mind).

------------------------------------------------------------------------

## 🚀 Features

-   Start a new game by selecting a game and adding named players
-   Enforces player limits per game
-   Track scores round-by-round or per hand
-   View running totals in real time
-   Reset or clear scores at any time
-   View history of previous games
-   Mobile-friendly and desktop-friendly UI

------------------------------------------------------------------------

## 🎯 Supported Games (MVP)

-   Spades
-   Casino (includes scoring cheat sheet)
-   Skyjo
-   Catch Five
-   Downforce

------------------------------------------------------------------------

## ⚠️ Notes

-   Some games include simplified or default scoring rules
-   Many games have house-rule variations; these are not fully
    configurable yet
-   Scoring logic is designed to be extendable in future updates
-   Yes, this was vibe coded, get over it.  It does not do anything important or store any critical data.
